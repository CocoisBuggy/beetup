# Beetup

Android fitness tracking app for logging exercises and workouts.

## What it does
- Create custom exercises with different measurement types
- Log workouts with resistance values (weight, reps, duration, distance)
- Track exercise history and view activity patterns
- Room database for local data storage
- Ability to export data for use anywhere
- **NO ONLINE ACCOUNTS**

## Tech stack
- Kotlin + Jetpack Compose
- Room database (v2)
- Material Design 3

## Build
```bash
./gradlew assembleDebug
```
